### 使用：

1、项目根目录build.gradle文件添加maven地址：
```ruby
allprojects {
    repositories {
        google()
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}
```

2、需要使用的RoundView的模块的build.gradle文件添加：
```ruby
dependencies {
    implementation 'com.github.shengqf:RoundView:1.0.0'
}

```

3、同步工程即可使用
